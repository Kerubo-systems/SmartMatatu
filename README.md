# SmartMatatu

SmartMatatu is an independent engineering research project exploring how technology can improve safety, accountability, comfort, and sustainability in Kenyan public transportation.

## Current Version

Version 1.0

## Features

- Passenger Experience Survey
- Mobile-first design
- Anonymous responses
- Accessible interface
- Modular survey engine

## Future Plans

- Flask backend
- PostgreSQL database
- Admin dashboard
- Data analytics
- AI-assisted insights
